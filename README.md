# 대빵이🍞
**빵의 도시 대전의 빵집 정보를 한눈에 알아볼 수 있는 사이트**

<br />

  **내용**
  
 - 대전의 다양한 빵집들을 소개하고, 빵을 사랑하는 빵순이, 빵돌이끼리 댓글로 소통할 수 있습니다.
 - 좋아요 기능을 통해 나만의 빵집 리스트를 만들 수 있습니다.
